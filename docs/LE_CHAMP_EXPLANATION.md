# Le Champ — Explication du Fonctionnement

## 📍 Comment ça marche

### Les traces dans Le Champ

**Le Champ affiche les inscriptions partagées de TOUS les utilisateurs** (pas les vôtres seulement).

**Fonctionnement :**
1. **Quand vous écrivez une inscription** (dans "Ma Carte" → Écrire)
2. **Vous pouvez cocher "Partager au Champ"** (checkbox `opt_in_field`)
3. **Si vous cochez** → Votre inscription apparaît dans Le Champ pour tous
4. **Si vous ne cochez pas** → Elle reste uniquement dans votre "Ma Carte"

### Différence avec "Ma Carte"

| | Ma Carte | Le Champ |
|---|---|---|
| **Contenu** | Vos inscriptions personnelles | Inscriptions partagées de tous |
| **Visibilité** | Privé (vous seul) | Public (tous les utilisateurs) |
| **Source** | Vous écrivez ici | D'autres utilisateurs ont partagé |
| **Anonymat** | Lié à votre carte | Anonyme (pas de card_id visible) |

### Comment partager une inscription

1. Allez dans **"Ma Carte"**
2. Cliquez sur **"Écrire"** (inscription)
3. Écrivez votre texte
4. **Cochez "Partager au Champ"** ✅
5. Enregistrez

→ Votre inscription apparaîtra dans Le Champ pour tous les autres utilisateurs.
