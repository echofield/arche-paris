# Canonical Artifact-to-Act Mapping

ARCHÉ canonical mapping between urban artifact types and valid existential acts.

## Act Set
- `Witness`
- `Consecrate`
- `Guard`
- `Testify`
- `Integrate`

## Invalid Pairing Rules

- `Consecrate` is invalid when artifact has no place-bound symbolic charge.
- `Guard` is invalid when no continuity risk or custodial obligation is present.
- `Testify` is invalid when act is only private logging with no public memory consequence.
- `Integrate` is invalid when no relation to existing trace lineage is established.

## Drift Safeguards (Mapping Level)

- Any mapping that privileges speed, score, or efficiency over place relation is invalid.
- Any mapping that turns artifacts into collectible loot is invalid.
- Any mapping that centers audience attention over urban reciprocity is invalid.
- Any mapping that reduces artifact meaning to tourism facts is invalid.
